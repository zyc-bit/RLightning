# Humanoid Toolkits

This package integrates the toolkit for humanoid motion data processing. The pipeline is organized as follows

1. download dataset
2. retargeting to a specified robots
3. run visualization to ensure the correctness

## Supported Robots

- Unitree G1
- Unitree H1

To add your own robot, please refer to xxxx.

## Installation

Firstly, we need to install the requirements with `uv pip install -r rlightning/humanoid/requirements.txt`. Then, choose one of registered dataset to download and running retargeting. You can directly run the following pipeline, then all things will be ready
